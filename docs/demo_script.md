# 2-Minute Executive Demo Script

## Goal
Show that we pull **real HR cost data from SAC**, produce a **credible 10-year baseline**, and use **AI to translate text into scenario parameters + rationale**.

## Setup (30 seconds)
1) Open the app.
2) Point to the **Data from SAC** card:
   - Provider name/ID
   - Last refresh timestamp
   - Row count + date range
3) Click **Refresh from SAC** (optional if already fresh).

## Baseline (30 seconds)
1) Say: “This is **HR Cost (monthly, currency)** with actuals + a 10-year baseline forecast.”
2) Point to the **baseline chart** and the KPI row:
   - Last actual (monthly)
   - Next 12 months (base)
   - Year‑10 annualized (base)

## Preset scenario (30 seconds)
1) Click **Trade War Downside**.
2) Say: “This applies a **-8% shock** for **12 months** and **-3 pp/yr** growth.”
3) Point to the **Scenario Comparison** chart and **Scenario delta** KPI.

## AI scenario assistant (30 seconds)
1) In Scenario Assistant, enter:
   - “AI productivity leap reduces headcount growth.”
2) Click **Suggest parameters**.
3) Call out the **Executive summary** + drivers.
4) Click **Apply suggestion** and show the chart update.

## Close (10 seconds)
“In under two minutes, we can show **real SAC data**, a stable baseline, and **AI‑assisted scenarios** with transparent assumptions.”
