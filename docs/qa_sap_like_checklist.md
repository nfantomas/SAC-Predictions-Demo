# SAP-like UI QA Checklist (fast pass)

- SAC connection visible and last refresh shown
- KPI tiles readable (EUR/month; 12M/5Y/10Y; FTE estimated) with clear deltas
- Main chart shows Actual, Plan/Baseline, Scenario with consistent colors/legend
- Quarterly chart available in expander; units labeled; FTE = end-of-quarter
- What-if Scenarios: presets obvious; AI assistant concise with rationale in expander
- Data provenance card present (provider, dataset, filters, dates, row count)
- Technical sections (model assumptions, advanced options) collapsed by default
- Labels use SAP wording; units consistent (EUR, FTE definition)
